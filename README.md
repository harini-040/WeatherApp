# Weather App

This is a simple React Weather App that shows real-time weather info for any city using the OpenWeatherMap API.  

## Live Demo
https://harini-040.github.io/WeatherApp

## How to run locally
1. Clone the repo: `git clone https://github.com/harini-040/WeatherApp.git`  
2. Go to the folder: `cd WeatherApp`  
3. Install dependencies: `npm install`  
4. Start the app: `npm start`
